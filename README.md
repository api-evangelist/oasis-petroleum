# oasis-petroleum

Profile for Oasis Petroleum in the API Evangelist network. Fortune F1000 (rank 969).

## Status: Defunct — Merged into Chord Energy

Oasis Petroleum no longer exists as an independent company. On **July 1, 2022**, Oasis Petroleum and Whiting Petroleum completed a merger of equals to form **Chord Energy Corporation** (NASDAQ: **CHRD**), creating one of the largest pure-play operators in the Williston Basin (Bakken / Three Forks).

The legacy `oasispetroleum.com` domain redirects to `chordenergy.com`.

## API Footprint

**Zero APIs.** Oasis Petroleum never operated a public developer program, portal, OpenAPI specification, SDK, or any machine-readable interface. As an upstream oil and gas E&P operator, its external digital surface was limited to a corporate website and SEC investor disclosures. The successor entity (Chord Energy) has likewise not published any public API.
